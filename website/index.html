By Klaus Wuestefeld, Rodrigo B. de Oliveira, Kent Beck and contributors

Byecycle is an auto-arranging dependency analysis plugin for Eclipse. Its goal is to make you feel sick when you see bad code and to make you feel happy when you see good code.


Installation

Byecycle requires Eclipse 3.1 running on Java 5.

Plugin veterans, this is the Byecycle update site: http://priki.org/svn/byecycle/trunk/updatesite

Plugin newbies, let us take you step-by-step.

If you have any trouble whatsoever installing or running Byecycle, please contact us on the discussion group below.


Low CPU Consumption

Byecycle's auto-layout algorithm runs in what we call "good citizen mode". It will use only up to 20% CPU and the more other tasks use the CPU, the less Byecycle will use it.
