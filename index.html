By Klaus Wuestefeld, Rodrigo B. de Oliveira, Kent Beck and contributors

After a compiler and a text editor, a code-structure viewer is the first thing you want. Byke is the requisite code-structure viewer for Eclipse.


Installation

Byke requires Eclipse 3.1 or newer running on Java 5 or newer.

Update site: http://wuestefeld.name/svn/byke/trunk/updatesite

Once installed, right-click on any source package > Show Dependencies

If you have any trouble installing or running Byke, please contact us on the discussion group below.


Low CPU Consumption

Byke's auto-layout algorithm runs in what we call "good citizen mode". It will use only up to 20% CPU and the more other tasks use the CPU, the less Byke will use it.
