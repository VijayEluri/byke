By Klaus Wuestefeld, Rodrigo B. de Oliveira, Kent Beck and contributors

Byecycle is an auto-arranging dependency analysis plugin for Eclipse. Its goal is to make you feel sick when you see bad code and to make you feel happy when you see good code.

Requirements

Java Runtime Environment 5.0 or superior
Eclipse 3.1 or superior


Installation

Download and unpack the byecycle package to your Eclipse home directory (files will be extracted to the plugins directory). Restart Eclipse.
Important - Sometimes Eclipse simply does not recognize a new plugin. Starting Eclipse once with the -clean option will make it flush its plugin cache and should solve the problem.
